# Rodar localmente

<div style={{textAlign: 'justify'}}>

&emsp;Siga os passos abaixo para rodar o site de documentação no seu computador.

## Pré-requisitos

&emsp;Tenha instalado:
- [Node.js 18+](https://nodejs.org/)
- [Git](https://git-scm.com/)

## Passo a passo

**1.** Acesse a pasta do site de documentação:

```bash
cd docs-site/
```

**2.** Instale os pacotes necessários:

```bash
npm install
```

**3.** Inicie o servidor de desenvolvimento:

```bash
npm run start
```

**4.** Acesse no navegador:

```
http://localhost:3000/
```

&emsp;O servidor atualiza automaticamente quando você salva alterações nos arquivos `.md`.

</div>
